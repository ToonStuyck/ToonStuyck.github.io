Raytracer in Java

This project was part of the course Computer graphics given by professor P. Dutré.
However, since this project is still running I won't be able to provide any code snippets at this moment. Once the project is over I will make my code public on bitbucket.
The aim of this project was to write your own renderer in Java. We had to choose between acceleration methods or global illumination. I opted for global illumination.
